## 知识库采用mkdocs构建

* `mkdocs new [dir-name]` - 创建一个新项目
* `mkdocs serve` - 项目预览
* `mkdocs build` - 构建网站
* `mkdocs -h` - 输出项目信息

## 项目结构

    mkdocs.yml    # 配置文件
    docs/
        index.md  # 文件主页
